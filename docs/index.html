VACUA / 5×9
Principle Playground Alphabet Ligatures Use GitHub

An experimental modular display font. Each letter is drawn by the absence of vertical strokes on a 5 × 9 grid — never by their presence.

Generative · Modular · MIT

Belongs to the lineage of Wim Crouwel's New Alphabet (1967) and to the tradition of generative typography where the whole alphabet fits in a table.

The principle

A cell is conceptually filled with five vertical strokes. We remove the segments that the letter passes through. What stays drawn is the matter around the void; the void is the letter.

The trap, when you try to add a character: you do not draw the letter. You draw everything except the letter.

Converter

Plain text in — Vacua glyphs out. Type anything, pick a weight, then export the result as SVG (vector, infinite zoom) or PNG.

Weight
A–Z and "?" only · the top-left nub is the start-of-letter marker

The full alphabet

Angular letters (E, F, H, I, L, T) read very well. Round ones (C, O, Q) and diagonals (K, X, Z) are more suggested than drawn — inherent to a 5-column resolution, and it reinforces the cryptic feel. A few glyphs (I, T, Y, Z) are hand-tuned to break collisions.

Rotation ligatures

Two letters can be condensed: the second is the whole glyph rotated 90°. Its strokes become horizontal through the rotation, not by a redesign. Overlay a vertical and a horizontal letter → a woven grid. Recover the second letter by rotating the medium a quarter turn.

An orientation-based cipher: read one way, then the other. The geometric pitfall — the cell is 5×9, so the rotation needs a square 9×9 base with a centring offset.

Use

Install

# clone + editable install
python -m venv .venv && source .venv/bin/activate
pip install -e .

CLI

python -m vacua "HELLO" --weight bold -o specimen.png
python -m vacua --chart -o chart.png
python -m vacua "A" --ligature --rotate B -o lig.png
python -m vacua "VACUA" --scad -o out/vacua.scad

Python

from vacua import render_text, BOLD

img = render_text("VACUA", style=BOLD)
img.save("vacua.png")

OpenSCAD → 3D print

The scad/vacua_param.scad file is standalone — the whole alphabet is encoded inside. Edit the text variable, open in OpenSCAD, export to STL.

See docs/3d_printing.md for the full parameter list.
